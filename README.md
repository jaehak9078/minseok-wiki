# 👋 민석위키에 온 것을 환영한다!

{% hint style="info" %}
**민석위키에 대해서** : 민석위키는 민석이를 제외한 누구나 편집이 가능한 위키다. 상남자 민석에 대해 다루는 위키인만큼 반말을 국룰로 한다.
{% endhint %}
